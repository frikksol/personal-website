gpterm

This is a terminal client for chat-GPT written in rust. Buzzwords you say? YES! It works in either single prompt mode, or in conversation mode.

You can install it using Homebrew on mac, with Cargo, or build it from source. When I have the time I'll add Apt and AUR installation methods. You need a chat-GPT API key but that is easy to get and the application assists you in setting it up.

Not much more to say. Install it and have some fun!

Chat app

This is a realtime chat app which runs on a rust backend with rocket, diesel and a postgres database. The fronend is written in typescript using SvelteKit and Tailwind CSS. The backend is hosted on Render and the frondend is hosted on Vercel.

I wanted to start learning rust, since it seems like a really interesting language which can make fast and reliable code easier to write, and the fact that it runs on everything from web frontends with WebAssembly to embedded devices is very interesting to me. Also the cross-compilation in rust must be the best solution for cross-compilation I have ever seen.
